# VisarTid

A visual countdown timer with a clock-face UI. Set the minutes, start, and watch time count down on a classic clock dial with a red zone.

## Getting Started

```bash
nvm use          # switch to Node 22 via .nvmrc
npm install      # install dependencies
npm start        # serve on http://localhost:3000
```

Open `http://localhost:3000` in your browser.

## Always on Top

Click the pin button to open the timer in an always-on-top window. Requires Chrome 116+ or Edge 116+ (Document Picture-in-Picture API).

## Development

```bash
npm test         # run tests + lint
npm run lint     # lint only
npm run lint:fix # auto-fix lint issues
```
